# Blueprint
街づくり・生活ゲーム「Blueprint」です。現在開発中。

概要：<br>
Blueprintは自分の作った街で生活が出来るゲームである。<br>
また、住民として参加することも出来るので遊び方は幅広い。

名前の由来は設計図（Blueprint）であり、その名の通り建物などを設計から出来るのが特徴。
